# Dépôt des Concepts de Machine Learning

Ce dépôt est structuré en trois dossiers principaux : Mathématiques pour le Machine Learning, Supervised Learning et Unsupervised Learning, chacun contenant des ressources et exemples pertinents.

## Dossier : Mathématiques pour le Machine Learning

Ce dossier propose des ressources sur les mathématiques essentielles au machine learning, incluant la statistique, l'algèbre linéaire, le calcul différentiel et intégral, et la théorie des probabilités.

![Image Mathématiques pour ML](/images/math-ml.jpg)

### Concepts Clés
- Statistiques descriptives et inférentielles
- Algèbre linéaire
- Calcul différentiel et intégral
- Théorie des probabilités

### Ressources Utiles
- [Khan Academy - Math for ML](https://www.khanacademy.org/math)
- [3Blue1Brown - Essence of linear algebra](https://www.youtube.com/playlist?list=PLZHQObOWTQDPD3MizzM2xVFitgF8hE_ab)

## Dossier : Supervised Learning

Ce dossier couvre les techniques et théories derrière l'apprentissage supervisé, incluant divers modèles et frameworks populaires.

![Image Supervised Learning](/images/supervised-learning.jpg)

### Sujets Abordés
- **Classification** : Techniques de classification binaires et multiclass.
- **CNN (Convolutional Neural Networks)** : Réseaux de neurones convolutifs pour l'analyse d'image.
- **Deep CNNs** : Architectures avancées de CNNs pour des tâches complexes.
- **Error Analysis** : Techniques d'analyse des erreurs pour optimiser les modèles.
- **Keras** : Utilisation de Keras pour le prototypage rapide de modèles de deep learning.
- **Optimization** : Méthodes d'optimisation pour l'entraînement de réseaux de neurones.
- **Régularisation** : Techniques pour prévenir le surapprentissage.
- **Tensorflow** : Emploi de TensorFlow pour des modèles scalables et performants.
- **Transfert Learning** : Application du transfert de connaissances entre modèles.

### Ressources Utiles
- [Deep Learning with Python by François Chollet](https://www.manning.com/books/deep-learning-with-python)
- [TensorFlow Tutorial](https://www.tensorflow.org/tutorials)

## Dossier : Unsupervised Learning

Exploration de méthodes d'apprentissage non supervisé pour découvrir des structures cachées dans les données.

![Image Unsupervised Learning](/images/unsupervised-learning.jpg)

### Sujets Abordés
- **Autoencoder** : Modèles pour l'apprentissage de représentations efficientes.
- **RNNS (Recurrent Neural Networks)** : Réseaux de neurones récurrents pour les données séquentielles.
- **Clustering** : Techniques de regroupement pour identifier des sous-groupes dans les données.
- **Dimensionality Reduction** : Méthodes pour réduire le nombre de variables de données.
- **HMM (Hidden Markov Models)** : Modèles pour analyser les séquences de données.
- **Hyperparamètre Tuning** : Techniques pour l'optimisation des paramètres des modèles.

### Ressources Utiles
- [Scikit-learn - Unsupervised Learning](https://scikit-learn.org/stable/unsupervised_learning.html)
- [Deep Learning Book - Ian Goodfellow](https://www.deeplearningbook.org/)
